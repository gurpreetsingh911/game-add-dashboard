Game Launch Automation

This project automates the process of collecting game launch data and submitting it to a Google Form.

Features
- Scrapes game information from URLs
- Detects language using franc
- Automatically submits data to Google Forms
- Node.js backend using Express
- Axios for API requests
- Playwright automation

Tech Stack
- Node.js
- Express
- Playwright
- Axios
- franc (language detection)

How to Run
1. Clone repo
2. npm install
3. node server.js
